Generating the data

A. J. Rominger

08 May 2019

This vignette documents how to reproduce from raw source material the derived data associated with package warbler. Producing these data depends on functions defined in the warbler package so we first need to install and load that package:

devtools::install_github('ajrominger/wabler')
library(warbler)

Obtaining the Parulidae phylogeny

The Parulidae phylogeny (Lovette et al. 2010) used in this study is easy and quick to obtain through the Open Tree of Life (Hinchliff et al. 2015) and so is not included as data in the warbler packge. To obtain the phylogeny we simply query the Open Tree of Life through the rotl package:

library(ape)
library(rotl)

wtre <- get_study_tree(study_id = 'pg_2591', tree_id = 'tree6024')

Extracting expert range map polygons

Expert ranges were obtained from Bird Life International by requesting them here: http://datazone.birdlife.org/species/requestdis. As of the receiving date (22 November 2018) those data are provided as an ESRI geodatabase file. We have to match the taxonomy of the Lovette et al. (2010) phylogeny with the taxonomy of the Bird Life International maps. I do by first looking for complete matches, and then filling in unmatched taxa by hand by searching for known synonyms of the unmatched taxa until a match is found. The resulting synonymns table is included as data with the warbler package.

The function extractSppBirdLife extracts specific species of interest from the large database file provided by Bird Life International. The script inst/get_all_ranges.R extracts all the ranges of Parulidae and saves them to the data directory. It is reproduced below:

library(sp)
library(rgdal)

source('R/extractSppBirdLife.R')

syn <- read.csv('data/lovette_taxSyn.csv', as.is = TRUE)

extractSppBirdLife('~/Dropbox/Research/data/BirdLife_HBW_Bird_Maps/BOTW.gdb', 
                   '~/Dropbox/Research/warbler/data', 'botw_ranges', 
                   syn$botw_name, 
                   c(1, 2), mc.cores = 10)


wrange <- readOGR(path.expand('data/botw_ranges.shp'), 'botw_ranges')
wrange <- aggregate(wrange, by = list(name = wrange$SCINAME), FUN = paste, sep = '; ', 
                    dissolve = TRUE)
wrange@data <- wrange@data[, c('name', 'SISID', 'DATE_', 'SOURCE', 'COMPILER', 
                               'REVIEWERS', 'CITATION', 'VERSION')]
names(wrange) <- c('name', 'SISID', 'date', 'source', 'compiler', 'reviewers', 
                   'citation', 'version')

save(wrange, file = 'data/wrange.RData')
